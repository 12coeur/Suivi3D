Cet outil n’a pas d’autre but que de vous situer dans un relief. C’est pourquoi vous disposez d’un curseur « Relief Z ». S’il n’est pas calé à gauche, il triche, mais, vous le verrez cela peut servir. Le mode Winter peut s’améliorer avec le curseur « éclairage ». Pour les jours blancs ! . En mode Winter, vous aurez le plan des pistes. Certains navigateurs ne vous donnent pas accès au GPS , alors vous pouvez regarder avec un autre .
